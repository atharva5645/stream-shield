# Database Module

This folder contains the complete Mongoose database layer for the Antigravity backend.

## Structure
- \`config/\` - Connection and Mongoose setup
- \`models/\` - Mongoose compiled models
- \`schemas/\` - Mongoose schema definitions (with hooks and methods)
- \`repositories/\` - Database abstraction layer for CRUD operations
- \`validators/\` - Express validators for models
- \`aggregation/\` - Complex aggregation pipelines for analytics
- \`transactions/\` - Atomic multi-document operations
- \`seeders/\` - Database seeding scripts
- \`migrations/\` - Schema and indexing migration scripts
- \`constants/\` - Reusable constants (Roles, Statuses)
